laravel-español
===============

mensajes de alerta traducidos al español


##Instrucciones:

1. descargar la carpeta "es" y colocarla dentro de /application/language/

2. luego editar el archivo application.php en la carpeta config y cambia

        'language' => 'en',

      por 
  
        'language' => 'es',

3. y eso es todo
